Внести изменения в локальную рабочую копию
git commit -m "first commit"

Добавить новый удалённый Git-репозиторий под именем-сокращением, к которому будет проще обращаться (origin):
git remote add origin https://github.com/marat-landed/x-ray.git

Передать изменения в удаленный репозиторий
git push -u origin master

push - помещение данных в удаленный репозиторий
pull - получение данных из удаленного репозитория

origin - сервер

rm -rf /path/to/dir/*  — удалит только содержимое
